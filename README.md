# ***一键启动《Minecraft：Java版》V1.5最终版正式发布***

*目前自带1.12.2和2015愚人节版本*

*自带32位java环境，无需额外下载任何文件，抛弃繁琐的配置和卸载过程，启动方便，零基础电脑小白也可轻松使用*

***自带mod列表(限1.12.2)：optfine,JEL物品管理器,连锁挖矿***

**使用方法：下载后解压到U盘内，并双击“双击我启动”即可使用**
